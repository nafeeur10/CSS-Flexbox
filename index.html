Flexbox

Flexbox is a new layout module in CSS3 that lets you lay out, align and distribute space among items within a container, even when their sizes are unknown or dynamic.

Flexbox Properties

Property Name Property Description
flex-direction Specifies the direction of the flexible items inside a flex container
flex-wrap Specifies whether the flex items should wrap or not, if there is not enough room for them on one flex line
justify-content justify-content enables you to distribute free space that is left between the flex items on the main axis.
align-items align-items enables you to distribute free space that is left between the flex items on the cross axis.
